# garoma-project
This backend app is created as a Backend Engineer Coding Challenge by GAROMA

To Run, Execute The Following Commands In Order:

`npm install`

`npm run db:create`

`npm run db:migrate`

`npm run db:seed:all`

`npm run start`

To Run Tests, Use:

`npm run test`

`npm run test:clean` (If Running Tests For The First Time)
